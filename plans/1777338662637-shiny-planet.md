# 街機廳角色邀請系統計劃

## 目標
讓使用者可以從街機廳選擇角色、發送邀請到 chat，角色同意後出現在街機廳，並顯示圓形大頭貼進行對話。

---

## 一、系統架構

### 1.1 資料流
```
街機廳 (Arcade) 
    ↓ 選擇角色 + 發送邀請
Chat App (透過 postMessage)
    ↓ 角色回應 (AI 生成)
街機廳 (Arcade)
    ↓ 顯示角色大頭貼 + 對話
```

### 1.2 角色資料來源
- `sx_characters` - 角色列表
- `sx_masks` - 角色面具列表
- `sx_char_name`, `sx_char_avatar`, `sx_char_personality` - 當前角色設定

---

## 二、實現步驟

### Phase 1: 角色選擇 UI
**檔案**: `apps/arcade/arcade-invite.js` (新增)

1. 在街機廳 header 新增「邀請角色」按鈕
2. 點擊後顯示角色選擇 Modal
3. 讀取 `sx_characters` 和 `sx_masks` 顯示角色列表
4. 每個角色顯示大頭貼 + 名稱


### Phase 2: 邀請發送系統
**檔案**: `apps/arcade/arcade-invite.js`

1. 選擇角色後，發送 postMessage 到 chat
2. 邀請訊息格式：
```javascript
{
  type: 'ARCADE_INVITE',
  payload: {
    charId: 'xxx',
    charName: 'xxx',
    charAvatar: 'xxx',
    charPersonality: 'xxx',
    timestamp: Date.now()
  }
}
```

### Phase 3: Chat 端處理
**檔案**: `apps/chat/chat.js` (修改)

1. 監聽 `ARCADE_INVITE` 訊息
2. 使用 AI 生成角色回應
3. 顯示邀請卡片，讓使用者確認
4. 角色同意後，發送 `ARCADE_INVITE_ACCEPTED` 回街機廳

```javascript
// 在 chat.js 新增
window.addEventListener('message', (event) => {
  if (event.data.type === 'ARCADE_INVITE') {
    handleArcadeInvite(event.data.payload);
  }
});
```

### Phase 4: 角色大頭貼顯示

### Phase 5: 對話互動
**檔案**: `apps/arcade/arcade-avatar.js`

## 五、技術細節

### 5.1 postMessage 通訊
```javascript
// 街機廳 → Chat
window.parent.postMessage({
  type: 'ARCADE_INVITE',
  payload: { charId, charName, ... }
}, '*');

// Chat → 街機廳
window.parent.postMessage({
  type: 'ARCADE_INVITE_ACCEPTED',
  payload: { charData, accepted: true }
}, '*');
```

### 5.2 大頭貼位置
- 固定在 `position: fixed; bottom: 120px; right: 20px;`
- z-index 高於虛擬控制器
- 不遮擋遊戲區域

### 5.3 對話生成
- 使用現有的 AI API 配置
- 根據角色性格生成回應
- 支援多語言

## 七、用戶確認結果

1. **大頭貼位置**: 可拖曳 - 使用者可以自由拖曳到任意位置
2. **多角色**: 只邀請一個 - 簡單實現，一對一互動
3. **離開機制**: 兩者都要 - 手動點擊離開按鈕 + 離開街機廳自動離開

## 八、詳細實現方案

### 8.1 可拖曳大頭貼
```javascript
class ArcadeAvatar {
  constructor(charData) {
    this.charData = charData;
    this.isDragging = false;
    this.position = { x: window.innerWidth - 100, y: window.innerHeight - 200 };
  }
  
  setupDrag() {
    this.element.addEventListener('mousedown', (e) => this.startDrag(e));
    this.element.addEventListener('touchstart', (e) => this.startDrag(e));
    document.addEventListener('mousemove', (e) => this.drag(e));
    document.addEventListener('touchmove', (e) => this.drag(e));
    document.addEventListener('mouseup', () => this.endDrag());
    document.addEventListener('touchend', () => this.endDrag());
  }
}
```

### 8.2 離開機制
```javascript
// 手動離開
function leaveArcade() {
  if (arcadeAvatar) {
    arcadeAvatar.remove();
    arcadeAvatar = null;
  }
  localStorage.removeItem('sx_arcade_guest');
}

// 自動離開 (監聽 app 關閉)
window.addEventListener('beforeunload', () => {
  localStorage.removeItem('sx_arcade_guest');
});
```

### 8.3 狀態保存
```javascript
// 保存邀請的角色到 localStorage
const ARCADE_GUEST_KEY = 'sx_arcade_guest';

function saveArcadeGuest(charData) {
  localStorage.setItem(ARCADE_GUEST_KEY, JSON.stringify({
    ...charData,
    position: arcadeAvatar.position,
    joinedAt: Date.now()
  }));
}

function loadArcadeGuest() {
  const saved = localStorage.getItem(ARCADE_GUEST_KEY);
  return saved ? JSON.parse(saved) : null;
}
```
